# 图书管理系统简介 python379

## 项目概述

本图书管理系统基于Python的Flask框架开发，旨在为用户提供便捷的书籍管理、查看、评分和购物车功能，同时支持后台管理，确保系统的高效运行。

## 技术栈

- 后端：Python Flask
- 数据库：MySQL
- 前端：HTML/CSS/JavaScript

## 功能模块

### 用户端

- 用户登录注册
- 书籍查看
- 书籍详情页浏览
- 书籍评分
- 加入购物车
- 推荐书籍
- 历史评分查询
- 搜索功能

### 管理端

- 后台管理
- 购物车管理
- 可视化数据图表

## 系统角色

- 普通用户：可以查看、评分书籍，管理购物车等。
- 管理员：负责后台书籍管理、用户管理和系统监控。

## 运行环境

- 操作系统：兼容Windows/Linux/OS X
- Python版本：3.x
- 依赖管理：pip

## 部署步骤

1. 环境准备：安装Python 3.x、MySQL数据库、pip等。
2. 克隆或下载项目源码。
3. 安装依赖包：使用pip安装项目所需依赖。
4. 配置数据库：根据项目文档设置MySQL数据库。
5. 运行项目：使用Flask命令或开发服务器运行项目。
